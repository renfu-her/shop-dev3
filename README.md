## 使用方式
- 先建立 mysql 資料庫，並修改 .env 資料庫設定

```bash
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=admin-standard
DB_USERNAME=root
DB_PASSWORD=
```

- 修改 .env 資料庫設定後，安裝套件，建立資料表，新增預設資料

```bash
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan db:seed
```

### **預設是緑界測試環境，如果需要正式環境，請修改 .env 設定**
